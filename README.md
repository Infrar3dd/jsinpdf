# jsinpdf

An utility to embed JS to PDF file

Usage:

```python
jsinpdf.py [-h] [-i INPUT] [-o OUTPUT] [-js JAVASCRIPT]

Utility to embed JS to PDF

options:
  -h, --help            show this help message and exit
  -i, --input INPUT     Input PDF
  -o, --output OUTPUT   Output PDF
  -js, --javascript JAVASCRIPT File with JS code
```
